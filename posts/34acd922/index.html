What are LAMP and Shell Script

  1. LAMP 通常指的是 Linux, Apache, MySQL, PHP 用來開發 web 應用的組合包。
  2. Shell Script 是一份純文字檔，可以將多個指令寫在一起再一次執行，來達到自動化的目的。

Automatic LAMP installation with Shell Script

在 Ubuntu 22.04 一鍵安裝 LAMP + phpMyAdmin
在安裝過程中，有時候要手動輸入密碼，這時使用 debconf-set-selections 能預先定義好設定值，來避免手動配置。
基本使用方法: echo "package-name configuration-key type value" | sudo debconf-set-selections

  1. 配置 instal_lamp.sh

  2. 執行 install_lamp.sh
