安裝 Discord.js 及設置 Discord 機器人

Created2023-01-21|Updated2023-12-03|DevelopmentDiscord Bot
|Post View:

環境配置可參考：https://blog.garyhuang.tw/posts/745c7824/
注意 Node 版本大於 v16.9.0 即可

安裝 Discord.js

  1. 專案初始化
    創建一個專案目錄: discord-bot

    移至專案目錄下進行初始化，該指令會新增一個 package.js 檔，用來記錄專案相關資訊。

  2. 安裝 Discord.js

設置 Discord 機器人

  1. 前往 Discord Developer Portal 登入自己的帳號

  2. 點選右上角 “New Application” 按鈕，輸入機器人名稱，按下 Create

  3. 完成後可以看到以下畫面，在此可以編輯一些基本資訊

  4. 點選左側的 Bot 頁籤，再按下畫面右側的 Add Bot 新增機器人

  5. 按下 Yes, do it!

  6. 成功創建自己的 Discord bot 了，可以在此編輯機器人的頭像或名稱等資訊

把機器人加入 Discord server

  1. 點選左側 OAuth2 頁籤，點選 bot、applications.commands

  2. 最下方會生成一串 URL

  3. 點選 COPY 並輸入在網址列上，就能選擇要新增至哪個伺服器

  4. 成功新增機器人至 Discord server

References

https://discordjs.guide/#before-you-begin
