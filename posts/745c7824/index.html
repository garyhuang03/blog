Windows WSL 安裝 Ubuntu 及 Node.js

Created2023-01-14|Updated2023-12-03|DevelopmentDiscord Bot
|Post View:

Environment:
Windows 11、WSL 2、Ubuntu 20、Node.js 18

WSL install Ubuntu 20.04

ref: https://learn.microsoft.com/en-us/windows/wsl/about
WSL (Windows Subsystem for Linux): 讓使用者能夠直接在 Windows 上運行 GNU/Linux 環境。

  1. 在 Windows 搜尋列打上 Windows PowerShell 並以系統管理員身分執行。
  2. 安裝 WSL
  3. 查看可以下載的 Linux 版本
  4. 安裝 Ubuntu 20.04
  5. Ubuntu 相關設定
    會彈出一個 Ubuntu 的安裝視窗，並等待一下讓它安裝完成。

    設定使用者名稱、密碼，並進入 Ubuntu 環境

    查看 Ubuntu 版本

Ubuntu install Node.js 18

ref: https://discordjs.guide/preparations/
ref: https://github.com/nodesource/distributions#using-ubuntu-1
因為之後要用 Discord.js 做一個機器人，必須使用 Node v16.9.0 以上的版本，因此這邊安裝 v18.x 的版本。

  1. 更新 Ubuntu 套件
  2. 安裝 Node.js
  3. 查看 Node 版本

重新連進已安裝好的 Linux 版本

  1. 打開 Windows PowerShell 查看已安裝的 Linux 版本

  2. 運行指定的 Linux 版本
    -d 後面打上指定 Linux 版本的 Name，並輸入 whoami 確認是否為剛剛安裝及設定的 Ubuntu。

  3. 打上 exit 就可以退出

安裝其他版本的 Ubuntu

如果想安裝最新版的 Ubuntu 22，可以在 Microsoft Store 下載。
